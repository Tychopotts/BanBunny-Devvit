# Privacy Policy

**Ban Bunny** is a Reddit moderation tool that operates on the Devvit platform.

## Data Collection

This application collects and stores the following information when a user is banned from a subreddit where it is installed:

- Reddit username of the banned user
- Username of the moderator who issued the ban
- Ban reason and duration
- Subreddit name
- Timestamp of the ban

## Data Usage

- Ban records are stored in Reddit's Devvit Redis storage for logging purposes
- Ban notifications are sent to a Discord server configured by the subreddit's moderators via the HTTP Fetch API

## Data Sharing

No personal user information is shared beyond:
- The banned user's Reddit username
- The ban reason provided by the moderator

This information is sent only to the Discord webhook URL configured by the subreddit's moderation team.

## Data Not Collected

This application does **not** collect or store:
- Email addresses
- IP addresses
- Personal identifying information
- Any data beyond what is publicly visible in Reddit's moderation log

## Contact

For questions about this privacy policy, please contact the app developer or subreddit moderators.
